Chatzy is a real-time chat application that allows users to communicate seamlessly via private and group messages. Built using the modern MERN stack with Socket.IO for live interactions, Chatzy provides a clean, responsive UI and key social features like image sharing and group conversations.

🚀 Features
💬 One-on-One & Group Chat: Start private or group conversations with real-time updates.

📸 Image Upload: Share images directly in chat via drag-and-drop or file selection.

🟢 Real-Time Messaging: Powered by WebSockets using Socket.IO.

📱 Responsive UI: Optimized for both desktop and mobile views.

🔐 Authentication: Secure user login and session management.

⚙️ Scalable Architecture: Clean codebase with separate frontend and backend folders.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS

Backend: Node.js, Express.js

WebSockets: Socket.IO

Database: MongoDB (via Mongoose)

Image Handling: Cloudinary

Authentication: JWT-based login system
